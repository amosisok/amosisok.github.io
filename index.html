Kojobs

KoJobs is an online web app built for both employers and job seekers. Guest users can search for and apply to jobs for free! To post a job, users must create an employer account and be successfully verified by an admin.

The back-end utilizes Node.js for Salesforce API integration and React JS for the front-end. Bootstrap is also used for icons and styling.
